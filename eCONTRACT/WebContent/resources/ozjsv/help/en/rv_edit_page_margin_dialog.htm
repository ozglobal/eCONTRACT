Page Setup Dialog

 Top  Previous 

 

 

Page Setup dialog appears by clicking [Page setup] in the pulldown menu of Etc icon() on the toolbar.

 

Margin :

Set top and left page margin. To set right and bottom margin, check Auto adjust margin option

Unit :

Unit : measurement unit for margin

pt : point

inch : inch

cm : centimeter

Margin Option :

Use Default : set to default margin value

Auto adjust margin : auto-adjust page margin. If change margin when this option was checked, the report is auto-resized. Margins for all directions other than the changed direction are auto-adjusted.
