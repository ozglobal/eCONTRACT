Overview of OZ Report Viewer

 Top  Previous  Next

 

 

OZ Report Viewer is a client program that allows the end-user to view reports on the web. Various viewers are provided for diversity of user web environments.

Major features of OZ Report Viewer are as follows:

 

■Preview OZ reports on the web
■Save reports in various file formats (Microsoft Word/PowerPoint/Excel, PDF, TIFF, HTML, CSV, etc.)
■Edit a report or modify its properties
■Advanced printing features (handout printing and zoom in/out printing)
■Provides components allowing dynamic user input
■Search features for contents of reports
