MHTファイルで保存

 Top  Previous  Next

 

 

ツールバーで上書き保存アイコン()をクリックすると表示される保存ダイアログでファイル形式をWeb Page Save File(*.mht)で選択して[設定]ボタンをクリックすると、MHTファイルのオプションを設定するMHT保存オプションダイアログが表示されます。

 

保存方式

フォント名とフォントサイズを設定します。

フォント：ラベルのフォントを一括して変更します。保存フォントを使用をチェックした場合、各ラベルに設定されたフォントを無視してコンボボックス内で設定したフォントが全てのラベルに適用されます。

サイズ：ラベルのフォントサイズを一括して変更します。フォントサイズを設定した場合、各ラベルに設定されたフォントサイズを無視してコンボボックス内で設定したフォントサイズが全べてのラベルに適用されます。
