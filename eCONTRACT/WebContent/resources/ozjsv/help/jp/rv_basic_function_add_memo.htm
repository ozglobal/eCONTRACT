メモ追加

 Top  Previous  Next

 

 

レポートにメモを追加してテキストを入力するためにはメモ追加アイコン()をクリックして

 

レポートウィンドウで任意の位置にドラッグ&ドロップしてメモを描画します。

 

追加されたメモをダブルクリックするとテキストが入力できるようにテキストエディターが表示されます。

 

テキストエディターでテキストを入力した後、テキストエディター以外のレポート領域をクリックすると入力したテキストがメモに適用されます。

 

メモのフォント、塗りつぶし色、枠線などはメモを右クリックすると表示されるポップアップメニューで[編集]メニューをクリックすることで表示されるラベル書式ダイアログで変更可能です。
