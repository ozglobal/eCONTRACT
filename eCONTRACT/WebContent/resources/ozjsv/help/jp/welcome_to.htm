最上のレポートサービス環境をサポートするOZ Report Viewer

 

 

OZ Report ViewerはユーザーがWebを通じてレポートをプレビューし、プリンタに出力する機能を提供するOZ Report専用Viewerです。

 

OZ Report Viewerはユーザーが照会中のレポートの編集、属性変換、Microsoft(以下MS) Word、MS Powerpoint、MS Excel、PDF、TIFF、HTMLなど、多様なファイル形式での保存機能、割付け印刷、拡大/縮小出力などの印刷機能を提供します。

 

また、OZ Report Viewerは動的なユーザー入力コンポーネント機能とレポート検索機能を提供します。
