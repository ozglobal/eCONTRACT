OZDファイルで保存

 Top  Previous  Next

 

 

ツールバーで保存アイコン()をクリックすると表示される保存ダイアログでファイルの種類をOZ Report Data File(*.ozd)で選択して[設定]ボタンをクリックすると、OZDファイルのオプションを設定するOZD保存の設定ダイアログが表示されます。

 

パスワード

保存されるファイルにパスワードを設定します。

パスワードの確認

入力したパスワードを確認します。

編集内容を含む

プレビュー時のビューアに追加したメモ、コメント、図形（線、矢印、四角、円）、テキスト編集内容をOZDファイルへ含めて保存します。「編集内容を含む」オプションをチェックせず保存したOZDファイルを開くとメモと図形をOZDファイルに追加できないように関連アイコン（メモ、線、矢印、四角、円）が非活性化されます。ただし、e-Formビューアのみ対応しているコメント機能は該当オプションの設定と関係なく活性化になります。

複数レポートの保存

現在、開いている全てのレポートを１つのOZDファイルで保存します。

オプションをチェックしないと現在表示されているレポートのみ保存されます。

イメージを含む

イメージのパスがURL形式(http://、ozp://、file://)で設定された場合、該当イメージをOZDファイルに含めて保存します。イメージを含むオプションはイメージだけでなくてテキストを特定URL(http://、ozp://、file://)を通じて取得するように設定したノートにも適用されます。
