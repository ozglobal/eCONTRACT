页面导航

 Top  Previous  Next

 

 

单击工具栏的其他菜单图标 > 单击 [页面导航] 菜单。

在页面导航中单击 [查看报表结构] 后，按住 <Shift> 键点击所需的页图标。

会在选定页面图标上显示蓝色选中符号 ()，并显示选定页在行列以及整体结构中的位置。

 

 

选择多页

选择多页的方法有使用 <Shift> 键、拖放、双击等三种方法。

   <Shift> 键

按住 <Shift> 键选择所需的页图标。

 

   拖放

拖放选择所需的页图标。

 

   双击

双击页图标时全选整行的页。

 

 

在保存或打印范围选择选定页时，将会保存或打印显示蓝色选中符号的 () 页面。
